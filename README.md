Resumo do Projeto – Atividade Extensionista III - UNINTER

Este projeto, desenvolvido para a Atividade Extensionista III, é uma plataforma gratuita de educação financeira criada para ajudar pessoas que têm dificuldade em organizar suas finanças e começar a investir.

No site, o usuário encontra:

Um curso de finanças do básico ao avançado

Ferramentas úteis, como calculadora de juros e cotações de ações

Conteúdos simples e acessíveis para aprender a controlar gastos e iniciar investimentos.

O objetivo é tornar a educação financeira mais fácil, prática e acessível para todos.
